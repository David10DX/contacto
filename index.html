¿Cuánto cuesta una app?

El precio de una app dependerá del tiempo que nos lleve realizar cada una de las partes y la dificultad de este, no es lo mismo programar las mismas funciones en html5, css o código para aplicaciones móviles, ya sea para iOS, android o windows phone. Una vez tengamos la idea del cliente, con alguna web similar que te guste y todas las partes que llevaría, realizamos una cotización aproximada del precio. Te enviaremos todos los detalles desglosados por email, además de recomendaciones para mejorar tu sitio web con los costes que llevaría.
